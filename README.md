# Delivery Trust Intelligence: NexDrop Case Study
**## Project Objective
Evaluate whether premium subscriptions improve customer trust, ETA reliability, and overall satisfaction in hyperlocal delivery platforms.

## Tools Used
- Power BI
- SQL
- Python
- Excel

## Business Questions Answered

1. Do premium users experience better service?
2. Does ETA reliability affect customer satisfaction?
3. Which countries show higher delivery trust gaps?
4. How does weather influence delivery performance?
5. Are premium subscriptions increasing expectations without improving experience?

## Key Findings

- Premium users experienced slightly higher delays despite faster promised ETAs.
- Nearly 19% of premium experiences showed dissatisfaction risk.
- ETA failures influenced customer trust more strongly than delays alone.
- Premium memberships may increase expectations without improving operational reliability.

## Recommendations

1. Replace fixed ETA with dynamic ETA prediction.
2. Improve premium delivery reliability.
3. Trigger compensation after major delays.
4. Focus on trust recovery strategies.
